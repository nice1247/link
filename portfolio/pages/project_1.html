🌟 SOC 기술마켓 시스템 고도화 프로젝트

🔗 프로젝트 개요

구분 내용
**프로젝트 명** SOC 기술마켓 시스템 고도화
**역할** 중소기업의 기술/제품에 대한 공모, 마케팅, 소통 창구 제공 플랫폼 (현: www.techmarket.kr)
**참여 기간** 2022년 하반기 ~ 2023년 상반기

💻 담당 역할 및 기술 환경

공공/SI 환경에서 **공모 프로세스 아키텍처 기획**, **데이터 모델링** 및 전반적인 **백엔드 개발**을 담당했습니다. 보안 솔루션 연동과 외부 API 통합 개발을 주도하며 시스템의 기능 확장과 안정화에 핵심적인 역할을 수행했습니다.

기술 스택

분류 기술 상세
**프레임워크** eGovFrame (전자정부 표준 프레임워크), Java
**웹/서버** Apache, Tomcat, JavaScript
**데이터베이스** MsSQL (쿼리 최적화 및 관리)
**협업/버전 관리** GitLab
**주요 솔루션 연동** D'Amo (암복호화), 코난 테크 (통합 검색)

🏆 핵심 기여 및 성과

1. 복잡한 공모 프로세스 구조화 및 안정화

기존 단일 신청 방식에서 '추가 인증 및 신청자와 공모 간의 다중 연결 프로세스'를 포함하도록 공모 로직을 전면 개편했습니다.

[성과] 데이터 모델링(ERD 확장)을 통해 복잡하게 얽힌 비즈니스 흐름을 구조화하고, 트랜잭션의 안정성을 확보하여 데이터 정합성 오류를 최소화했습니다.

2. 보안 및 검색 성능 강화를 위한 솔루션 통합 주도

  • [보안 솔루션 연동] 개인 정보 보호 강화를 위해 암복호화 솔루션 (D'Amo)을 도입하고, 민감 정보 저장 시 암복호화 쿼리를 적용하는 로직을 백엔드 전반에 통합 개발했습니다.

    역할: 성능 저하 최소화를 위해 **쿼리 레벨 최적화 작업**을 진행했습니다.

  • [통합 검색 엔진 연동] 방대한 정보를 빠르고 정확하게 검색하도록 통합 검색 솔루션 (코난 테크)과의 연동 모듈을 설계 및 구현했습니다.

    성과: 풀 텍스트 검색 기능을 제공하여 사용자 경험을 개선하고, 백엔드 로드 부하를 검색 엔진으로 분산시켰습니다.
