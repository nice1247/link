3. 프레임워크 : eGovFrame (전자정부 표준 프레임워크), Java 웹/서버 : Apache, Tomcat, JavaScript 데이터베이스 : MsSQL (쿼리 최적화 및 관리) 협업/버전 관리 : GitLab 솔루션 연동 : D'Amo (암복호화 솔루션), 코난 테크 (통합 검색 솔루션)

4. - 복잡한 공모 프로세스 구조화 및 안정화 [프로세스 고도화] 기존 단일 신청 방식에서 **'추가 인증 및 신청자와 공모 간의 다중 연결 프로세스'**를 포함하도록 공모 로직을 전면 개편했습니다. 성과: 데이터 모델링(ERD 확장)을 통해 복잡하게 얽힌 비즈니스 흐름을 구조화하고, 트랜잭션의 안정성을 확보하여 데이터 정합성 오류를 최소화했습니다. - 보안 및 검색 성능 강화를 위한 솔루션 통합 주도 [보안 솔루션 연동] 개인 정보 보호 강화를 위해 **암복호화 솔루션(D'Amo)**을 도입하고, 민감 정보 저장 시 암복호화 쿼리를 적용하는 로직을 백엔드 전반에 걸쳐 통합 개발했습니다. 역할: 개발 단계에서 발생하는 성능 저하 문제를 최소화하기 위해 쿼리 레벨에서의 최적화 작업을 진행했습니다. [통합 검색 엔진 연동] 서비스 내 방대한 기술/제품 정보를 빠르고 정확하게 검색할 수 있도록 **통합 검색 솔루션(코난 테크)**과의 연동 모듈을 설계하고 구현했습니다. 성과: 풀 텍스트 검색 기능을 제공하여 사용자 검색 경험을 대폭 개선했으며, 백엔드 로드 부하를 통합 검색 엔진으로 분산시켰습니다. - 다양한 외부 서비스 API 통합 개발 [API 서비스 적용] 사용 편의성을 높이기 위해 알림톡, SMS, 이메일 등의 외부 메시징 서비스 API를 통합하고, 신뢰성 있는 정보 제공을 위한 공공데이터 및 크롤링 API 연동 모듈을 구현했습니다. 성과: 사용자의 참여 및 공모 진행 상황에 대한 실시간 알림 체계를 구축하여 서비스의 접근성과 사용자 유지율을 향상시켰습니다.

5. - 복잡한 비즈니스 로직 처리 경험 : 단순 CRUD를 넘어, 여러 단계의 검증 및 다중 데이터 연결이 필요한 복잡한 공모 프로세스를 논리적으로 설계하고 정리하는 심도 있는 경험을 했습니다. - 외부 연동 전문성 확보 : 암복호화, 통합 검색 등 핵심 솔루션과 다양한 외부 API를 성공적으로 통합함으로써, 실무에서 요구되는 외부 서비스 연동 역량을 효과적으로 입증했습니다. - 운영 노하우 : 클라이언트와의 지속적인 피드백 공유 및 요구사항 반영을 통해 프로젝트의 완성도를 높였으며, 실질적인 운영 환경에 적합한 백엔드 시스템 구현 능력을 강화했습니다.
