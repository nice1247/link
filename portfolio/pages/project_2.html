프로젝트 개요

청년 카페에서 사용되는 마일리지 기반 결제 시스템으로, 복잡한 오프라인 결제 과정을 단순화하고 실시간 마일리지 잔액 확인 및 차감 트랜잭션의 무결성을 확보했습니다.

  • 마일리지 결제 시스템 핵심 로직 구축 (Ver 1.0)
    Payment

    카페 점주와 사용자 간의 QR 코드 인증을 통한 마일리지 결제 API 시스템을 구현. 복잡한 오프라인 결제 과정을 단순화하고, 실시간 마일리지 잔액 확인 및 차감 트랜잭션의 무결성을 확보했습니다.

  • 사용자 트래픽 대응 및 쿼리 성능 최적화
    Performance

    연간 사용자 수 증가 (800명 → 1,500명)에 따른 정산/통계 페이지 지연 현상을 해결하기 위해 쿼리 최적화 작업을 진행. 불필요한 구문 삭제 및 대용량 데이터 처리 로직 개선을 통해 정산 통계 페이지의 로딩 시간을 3초에서 체감상 바로 볼 수 있도록 단축시키며 운영 안정성을 확보했습니다.

  • 보안 강화 및 결제 무결성 증대 (Ver 2.0)
    Security

    사용자의 악의적인 접근 및 결제 오용 가능성을 원천 차단하기 위해 결제 인증 방식을 OTP (One-Time Password) 방식으로 전환하는 작업을 수행. 매번 고유한 토큰을 사용하는 OTP 로직을 통합함으로써, 기존 결제 방식의 보안 취약점을 해결하고 결제 트랜잭션의 무결성과 속도를 동시에 향상시켰습니다.

  • 앱 버전 관리를 위한 유연한 API 설계
    Management

    모바일 앱 서비스의 특성을 고려하여, 신규 기능 및 보안 패치가 적용된 버전을 강제/선택 업데이트할 수 있는 유연한 버전 관리 로직을 백엔드에 구축했습니다.

프로젝트를 통해 얻은 역량

  • 트래픽 기반의 최적화 경험
    실제 운영 환경에서 성능 병목 현상을 체감하고, 쿼리 레벨에서 해결하는 실무 경험을 쌓았습니다.
  • 보안과 사용성 균형
    마일리지/결제 기능에서 보안 무결성(OTP 도입)과 사용자 편의성 사이의 균형점을 찾는 시스템 설계 능력을 강화했습니다.
  • 앱 운영 노하우
    안정적인 서비스 제공을 위한 앱 버전 관리 체계를 직접 설계하고 구축하는 경험을 했습니다.
